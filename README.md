## Deskripsi

Mata kuliah ini dirancang untuk mengembangkan kemampuan bahasa Inggris mahasiswa dalam konteks Teknologi Informasi pada level upper intermediate (CEFR B2). Kurikulum terintegrasi mencakup empat keterampilan utama â€” Listening, Speaking, Reading, dan Writing â€” dengan penekanan pada kosa kata teknis dan komunikasi profesional di bidang IT.

Topik pembelajaran mencakup Computer Systems, Hardware, Operating Systems, Software Development, Networking, Cybersecurity, Cloud Computing, Web Development, Databases, Mobile Development, AI/ML, Data Science, IT Project Management, dan Emerging Technologies.
