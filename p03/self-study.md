## B. GRAMMAR SELF-STUDY

### B1. Grammar Summary: Present Perfect

#### Form

**Positive:** Subject + have/has + past participle
**Negative:** Subject + have/has + not + past participle
**Question:** Have/Has + subject + past participle?

#### Use

1. **Completed action with present relevance:** "I have installed the software." (It's ready to use now)
2. **Experience up to now:** "Have you ever used Linux?" (At any time in your life)
3. **Recent action:** "The system has just finished updating." (Very recently)
4. **Unfinished time period:** "I have updated three computers today." (Today isn't over)

#### Time Expressions

| With Present Perfect | NOT with Present Perfect |
|---------------------|--------------------------|
| already, yet, just | yesterday, last week |
| ever, never | in 2020, two days ago |
| recently, lately | when I was young |
| so far, up to now | at 3 o'clock |
| this week/month/year | on Monday |

#### IT Examples

- "I have configured the network settings." (Done, relevant now)
- "Have you installed the latest driver?" (Asking about completion)
- "She hasn't enabled TPM yet." (Still needs to be done)
- "We have deployed the update to all servers." (Completed action)

---

### B2. Grammar Summary: Imperatives

#### Form

**Positive:** Base verb (no subject)
**Negative:** Don't/Do not + base verb

#### Use

1. **Instructions:** "Click on the Settings icon."
2. **Commands:** "Restart your computer now."
3. **Advice:** "Always back up your data."
4. **Warnings:** "Don't turn off the computer during installation."
5. **Requests (polite):** "Please wait while the system updates."

#### Sequencing Words for Instructions

| First Step | Middle Steps | Final Step |
|------------|--------------|------------|
| First, ... | Then, ... | Finally, ... |
| To begin, ... | Next, ... | Lastly, ... |
| Start by ... | After that, ... | To finish, ... |

#### IT Examples

- "First, download the installation file."
- "Next, run the setup wizard."
- "Don't disconnect the USB during installation."
- "Finally, restart your computer to complete the setup."

---

### B3. Grammar Summary: Modals for Instructions

#### Must / Have to (Strong Obligation)

- "You must restart after installing the update."
- "You have to enable TPM for Windows 11."

#### Should / Ought to (Advice/Recommendation)

- "You should back up your files before upgrading."
- "You ought to check the system requirements first."

#### Can / Could (Possibility/Ability)

- "You can install the software from the USB drive."
- "Could you check if the driver is compatible?"

#### Need to (Necessity)

- "You need to enter your product key."
- "The system needs to restart to apply changes."

## E. WRITING TASK (Homework)

### Assignment: Step-by-Step Installation Guide

**Sub-CPMK:** LO3.2 (Instructions), LO3.3 (Technical Documentation)

#### Task Description

Write a step-by-step installation guide (200-250 words) for ONE of the following:
- **Option A:** Installing a software application (e.g., VS Code, Chrome, Python)
- **Option B:** Installing or setting up a component (e.g., a printer driver, antivirus)
- **Option C:** A basic OS configuration task (e.g., setting up user accounts, configuring firewall)

#### Requirements Checklist

Your guide MUST include:

| Requirement | Check |
|-------------|-------|
| Clear title stating what will be installed/configured | ☐ |
| Introduction explaining the purpose (1-2 sentences) | ☐ |
| System requirements or prerequisites | ☐ |
| At least 5 numbered steps | ☐ |
| At least 5 imperative sentences | ☐ |
| At least 2 Present Perfect sentences | ☐ |
| At least 2 modal verbs (must, should, can, need to) | ☐ |
| Minimum 5 OS/installation vocabulary terms | ☐ |
| A closing statement confirming completion | ☐ |
| 200-250 words | ☐ |

#### Guide Structure

**Title:** How to Install/Configure [Name]

**Introduction (1-2 sentences)**
- What you're installing
- Why it's useful

**Prerequisites/Requirements (2-3 sentences)**
- System requirements
- What the user needs before starting

**Steps (5-8 numbered steps)**
- Use imperatives
- Be clear and specific
- Include any warnings

**Conclusion (1-2 sentences)**
- Confirm successful installation
- Optional: Mention next steps

#### Model Answer

> **How to Install Visual Studio Code**
>
> Visual Studio Code is a popular code editor used by developers worldwide. This guide will help you install VS Code on your Windows computer.
>
> **Before You Begin:**
> You need to have Windows 10 or later installed on your computer. Make sure you have at least 500 MB of free disk space. You should also have administrator privileges.
>
> **Installation Steps:**
>
> 1. First, open your web browser and go to https://code.visualstudio.com.
> 2. Click the "Download for Windows" button. The installer will download automatically.
> 3. Once the download has finished, open the installer file (VSCodeSetup.exe).
> 4. Accept the license agreement and click "Next."
> 5. Choose your installation location or keep the default. Click "Next."
> 6. Select additional options (add to PATH, create desktop icon). Click "Next."
> 7. Finally, click "Install" and wait for the installation to complete.
> 8. Click "Finish" to launch VS Code.
>
> You have successfully installed Visual Studio Code! You can now start coding by opening a folder or creating a new file.

*Word count: 178 words*

**Note:** The model answer is slightly shorter than required. Your guide should be 200-250 words with more details.
