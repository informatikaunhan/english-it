## B. GRAMMAR SELF-STUDY

### B1. Grammar Summary: Past Simple, Past Continuous, and Passive Voice

#### Past Simple

**Form:**
- Regular: Subject + verb-ed (worked, developed, tested)
- Irregular: Subject + irregular past form (wrote, built, found)
- Negative: Subject + did not + base verb
- Question: Did + subject + base verb?

**Use:**
1. **Completed actions:** "The team completed the project last month."
2. **Sequential events:** "First, we gathered requirements. Then, we designed the system."
3. **Past facts:** "The original version used waterfall methodology."

**IT Examples:**
- "The developer **wrote** the code in Python."
- "We **implemented** the new feature last sprint."
- "Testing **revealed** three critical bugs."

---

#### Past Continuous

**Form:**
- Subject + was/were + verb-ing

**Use:**
1. **Actions in progress at a specific past time:** "At 3 PM, I was debugging the code."
2. **Background actions interrupted by another event:** "While I was testing, the server crashed."
3. **Two simultaneous past actions:** "The frontend team was building the UI while the backend team was writing APIs."

**IT Examples:**
- "While we **were working** on authentication, we found a bug."
- "The system **was processing** data when the error occurred."
- "I **was reviewing** code when my manager called."

---

#### Passive Voice (Past)

**Form:**
- Subject + was/were + past participle (+ by agent)

**Use:**
1. **Focus on the action/result:** "The code was reviewed by senior developers."
2. **Unknown or unimportant agent:** "A bug was discovered during testing."
3. **Formal/technical writing:** "The system was designed using microservices architecture."

**IT Examples:**
- "The application **was built** using React Native."
- "All requirements **were documented** in the specification."
- "The vulnerability **was fixed** within 48 hours."
