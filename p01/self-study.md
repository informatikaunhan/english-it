## B. GRAMMAR SELF-STUDY

### B1. Grammar Summary: Present Simple vs. Present Continuous

#### Present Simple

**Form:**
- Positive: Subject + base verb (+s/es for he/she/it)
- Negative: Subject + do/does + not + base verb
- Question: Do/Does + subject + base verb?

**Use:**
1. **Regular/habitual actions:** "I check my emails every morning."
2. **Permanent situations:** "She works at Google."
3. **General truths/facts:** "A firewall protects the network."
4. **Job descriptions:** "A software developer writes code."

**Time expressions:** always, usually, often, sometimes, every day/week/month, normally

**IT Examples:**
- "The server processes thousands of requests per minute."
- "I specialize in cloud computing."
- "Our team develops mobile applications."

---

#### Present Continuous

**Form:**
- Positive: Subject + am/is/are + verb-ing
- Negative: Subject + am/is/are + not + verb-ing
- Question: Am/Is/Are + subject + verb-ing?

**Use:**
1. **Actions happening now:** "I'm troubleshooting a bug right now."
2. **Temporary situations:** "She's working from home this week."
3. **Current projects:** "We're developing a new feature."
4. **Changing situations:** "Technology is evolving rapidly."

**Time expressions:** now, right now, at the moment, currently, this week/month, today

**IT Examples:**
- "I'm currently working on a security audit."
- "Our team is migrating data to the cloud."
- "She's learning Python this semester."

---

#### State Verbs (NOT used in continuous form)

These verbs describe states, not actions. Use Present Simple, not Continuous:

| Category | Verbs |
|----------|-------|
| **Mental states** | know, understand, believe, think (= opinion), remember, forget |
| **Emotions** | like, love, hate, prefer, want, need |
| **Possession** | have, own, belong, possess |
| **Senses** | see, hear, smell, taste, feel (= sensation) |
| **Other** | seem, appear, contain, consist, mean |
